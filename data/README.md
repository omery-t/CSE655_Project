# Dataset: Survey Lung Cancer

## Source
This dataset is known as the "survey lung cancer" dataset.

## Description
- **Samples**: 309 records
- **Features**: 15 features + 1 target variable
- **Target**: LUNG_CANCER (YES/NO - binary classification)

## Features
| Feature | Description | Values |
|---------|-------------|--------|
| GENDER | Gender of the patient | M (Male), F (Female) |
| AGE | Age of the patient | Integer |
| SMOKING | Smoking habit | 1 (No), 2 (Yes) |
| YELLOW_FINGERS | Yellow fingers symptom | 1 (No), 2 (Yes) |
| ANXIETY | Anxiety level | 1 (No), 2 (Yes) |
| PEER_PRESSURE | Peer pressure influence | 1 (No), 2 (Yes) |
| CHRONIC DISEASE | Chronic disease presence | 1 (No), 2 (Yes) |
| FATIGUE | Fatigue symptom | 1 (No), 2 (Yes) |
| ALLERGY | Allergy presence | 1 (No), 2 (Yes) |
| WHEEZING | Wheezing symptom | 1 (No), 2 (Yes) |
| ALCOHOL CONSUMING | Alcohol consumption | 1 (No), 2 (Yes) |
| COUGHING | Coughing symptom | 1 (No), 2 (Yes) |
| SHORTNESS OF BREATH | Shortness of breath | 1 (No), 2 (Yes) |
| SWALLOWING DIFFICULTY | Swallowing difficulty | 1 (No), 2 (Yes) |
| CHEST PAIN | Chest pain symptom | 1 (No), 2 (Yes) |
| LUNG_CANCER | Target variable | YES, NO |

## Notes
- No missing values in the original dataset
- No outliers present
